# MediFix - Masaka Hospital

![Masaka Hospital Logo](MediFix/assets/images/logos/masaka-logo.png)

MediFix is a sleek, modern medical equipment maintenance and troubleshooting platform designed specifically for **Masaka Hospital**. 

## Features
- **Equipment Tracking**: Dedicated dashboard for monitoring medical device status.
- **Troubleshooting Support**: Integrated AI assistant to help technicians diagnose and fix issues faster.
- **Modern UI/UX**: Premium dark/light mode interface with glassmorphism aesthetics.
- **Seamless Management**: Streamlined workflows for doctors and hospital administrators.

## Branding
This repository has been fully rebranded for **Masaka Hospital**, including all logos, footer text, and UI elements.
